# 🚗 EV Resale Price Prediction

## 📌 Overview

This project predicts the resale price of electric vehicles using Machine Learning and battery degradation modeling.

---

## ⚙️ Tech Stack

* Python
* Flask
* Scikit-learn
* Pandas, NumPy
* HTML, CSS

---

## 🧠 Features

* Predict resale price based on:

  * Brand, Model, Variant
  * Purchase & resale year
* Battery degradation modeling (~2% yearly)
* Shows battery health & range over time
* Interactive web interface

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
python app.py
```

Then open: http://127.0.0.1:5000

---

## 📂 Project Structure

* app.py → Flask app
* train_pipeline.py → Model training
* model.pkl → Trained model
* templates/ → Frontend UI
